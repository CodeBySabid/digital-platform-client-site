# 🚀 Key Features of Digital Platform

## 🔐 Authentication & Authorization
- Secure user authentication system.
- Login & registration using Firebase authentication.
- Role-based access control (Admin / User).
- Protected routes for private dashboard pages.

## 👤 User Profile Management
- Users can create and manage their profile.
- Profile information including name, email, role, and account creation date.
- Profile image support.

## 📊 Admin Dashboard
- Admin can manage users from a dedicated dashboard.
- Ability to view all registered users.
- Role management and user control.

## 📦 Data Management with Backend API
- RESTful API built with Node.js and Express.
- Secure server communication using Axios.
- MongoDB database integration for storing user and platform data.

## ⚡ Dynamic Data Fetching
- Efficient data fetching using React Query.
- Real-time UI updates when data changes.
- Optimized API calls with caching.

## 🎨 Responsive UI Design
- Fully responsive design for mobile, tablet, and desktop.
- Modern UI built with Tailwind CSS.
- Clean and user-friendly interface.

## 🔒 Secure API Handling
- Axios interceptor for secure API requests.
- Token-based request handling.
